## 4. COMPETITOR DEEP-DIVES

### 4.1 Scrivener
**What it is:** The 20-year-old gold standard. A desktop-first writing environment originally built for novelists and screenwriters but extended to academics. One-time purchase software.

**Strengths:**
- Binder-based project organization (folders within folders = ultimate flexibility)
- Corkboard view (index cards for scenes — visual plotters love it)
- Split-screen editing (write + research side by side)
- Compile feature — immensely powerful export with custom formatting per section type
- Templates for novels, screenplays, academic papers, poetry
- Full-screen composition mode
- Target word counts per document
- Research folder — import PDFs, images, web pages into the same project
- Snapshot system (version history per chapter)

**Weaknesses:**
- Ancient, dense UI — enormous learning curve. New users give up.
- Zero real-time AI integration
- No collaboration (single-player only)
- No cloud-native — requires Dropbox for sync (archaic)
- No web version — Mac/Windows/iOS only
- No grammar/style tools built in
- No character relationship maps
- Paid per platform ($59.99 Mac + $59.99 Windows separately)

**Pricing:** $59.99 one-time (Mac or Windows), $23.99 iOS
**Target:** Novelists, academics, screenwriters

**What Editorial Sanctuary can beat it on:** Real-time AI, cloud-native, collaboration, modern UX, universality across genres, web access from any device.

### 4.2 Sudowrite
**What it is:** The most sophisticated AI writing assistant for fiction. Built by fiction writers *for* fiction writers. Subscription SaaS.

**Strengths:**
- Muse AI model — trained specifically on literary fiction, understands scene blocking, dialogue, subtext
- Story Bible — structured context that AI actively references when generating
- Story Engine — can generate an entire book from concept to outline to draft
- Brainstorming tools — dialogue, characters, worldbuilding, plot points, names, places
- Rewrite tool — "Show, don't tell," "More conflict," "More sensory detail" modes
- Describe tool — expands description using multiple senses + metaphors
- Guided Write — user inputs direction, AI writes ~500 words toward it
- Character and worldbuilding generators
- Visualize feature — generates reference images from manuscript text

**Weaknesses:**
- 100% fiction-centric — useless for academic, technical, non-fiction, songwriting
- No citation manager, no equations, no code blocks
- Expensive at $29-129/month for word credit limits
- No offline mode
- No real export formatting (DOCX basic only)
- No collaboration
- No chapter structure management tools (no planning, just writing)
- No reading level meter
- Gets "verbose and flowery" — often over-generates purple prose

**Pricing:** $19/mo (Hobby), $29/mo (Pro), $129/mo (Max) — credit-based
**Target:** Fiction novelists, short story writers

**What Editorial Sanctuary can beat it on:** Every non-fiction writer (they have zero competition here), universal genre support, pricing (no credit limits), integrated project structure + AI (not just an AI bolted onto a text box), export quality.
