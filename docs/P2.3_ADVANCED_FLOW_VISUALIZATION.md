## Components to Build

### 1. Gantt Chart Timeline (`FlowGanttChart`)

**Purpose:** Visualize events across timeline with duration, dependencies, and status

**Features:**
- Horizontal bar chart: x-axis = timeline (days/months), y-axis = events
- Color-coded by status: planned (gray), in_progress (blue), completed (green)
- Dependency lines between related events
- Hover tooltip: event title, dates, dependencies
- Drag to resize event duration
- Click to open event editor
- Responsive: scale timeline by zoom level

**Data Requirements:**
- `FlowEvent[]` with timeline_position, duration, status
- `FlowDependency[]` for drawing connector lines

**Props:**
```typescript
interface FlowGanttChartProps {
  events: FlowEvent[];
  dependencies: FlowDependency[];
  onEventClick: (eventId: string) => void;
  onEventResize?: (eventId: string, newDuration: number) => void;
  zoom?: 'week' | 'month' | 'quarter';
}
```

---

### 2. Event Filter Bar (`FlowEventFilterPanel`)

**Purpose:** Filter events by type, status, date range, and search

**Features:**
- Type filter: multi-select dropdown with project-type-aware event types
- Status filter: checkboxes (planned, in_progress, completed, archived)
- Date range picker: start date, end date
- Search input: quick search across title + description
- Clear all button
- Active filter badge count

**Persistence:**
- Save filter state to URL search params (`?type=scene&status=planned&dateFrom=...`)
- Restore on page load

**Props:**
```typescript
interface FlowEventFilterPanelProps {
  eventTypes: string[];
  onFilter: (criteria: FilterCriteria) => void;
  activeFilters?: FilterCriteria;
}

interface FilterCriteria {
  types?: string[];
  statuses?: ('planned' | 'in_progress' | 'completed' | 'archived')[];
  dateRange?: { from: Date; to: Date };
  searchQuery?: string;
}
```

### 3. Bulk Operations Toolbar (`FlowBulkOperationsBar`)

**Purpose:** Select multiple events and perform batch operations

**Features:**
- Checkbox column in event list for multi-select
- Toolbar appears when items selected (sticky top)
- Actions:
  - Change status (dropdown: planned → in_progress → completed)
  - Add/remove tags
  - Delete selected (with confirmation)
  - Export selected
- Selection counter: "5 selected"
- Select all / clear all options

**UI Pattern:**
```
[☑ 5 selected] [Status: ▼] [Tags: ▼] [Delete] [Export] [✕ Clear]
```

**API Integration:**
```typescript
// Batch update endpoint (if needed)

  event_ids: string[];
  updates: Partial<FlowEventUpdateRequest>;
```

### 5. Timeline Analytics (`FlowAnalyticsPanel`)

**Purpose:** Display insights about project pacing and event distribution

**Metrics:**
- **Event Count:** Total events by status (breakdown pie chart)
- **Timeline Span:** Days/weeks from first to last event
- **Average Duration:** Avg event length
- **Busiest Period:** Time range with most events
- **Pacing Chart:** Bar chart showing events per week/month
- **Status Distribution:** Pie chart (% planned, in_progress, completed)
- **Dependency Density:** % events with dependencies

**Visualization Libraries:**
- Recharts for charts (lightweight, React-friendly)
- Lucide-react for icons

**Props:**
```typescript
interface FlowAnalyticsPanelProps {
  events: FlowEvent[];
  dependencies: FlowDependency[];
}
```
