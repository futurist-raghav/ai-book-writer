## What Was Built This Session

### Backend (3 files)
1. **Migration 011** (`alembic/versions/011_bibliography.py`)
   - `bibliography` table: id, book_id, title, authors, year, source_url, source_type, citation_formats (JSON), notes
   - `chapter_citations` table: junction linking chapters to sources with page numbers, context
   - Indexes on book_id, chapter_id, bibliography_id
   - Unique constraint on (chapter, bibliography, offset)

2. **API Router** (`app/api/v1/bibliography_router.py`)
   - CRUD endpoints for Bibliography
     - `GET /books/{book_id}/bibliography` - List all sources
     - `POST /books/{book_id}/bibliography` - Create source
     - `GET /books/{book_id}/bibliography/{source_id}` - Get details
     - `PATCH /books/{book_id}/bibliography/{source_id}` - Update source
     - `DELETE /books/{book_id}/bibliography/{source_id}` - Delete source
   - Citation endpoints
     - `GET /chapters/{chapter_id}/citations` - List chapter citations
     - `POST /chapters/{chapter_id}/citations` - Add citation (with page #, context)
     - `DELETE /chapters/{chapter_id}/citations/{citation_id}` - Remove citation
   - AI endpoints (stubbed for next phase)
     - `POST /chapters/{chapter_id}/generate-bibliography` - Suggest citation locations
     - `POST /bibliography/generate-from-text` - Extract citations from text

### Frontend (3 files)
1. **BibliographyManager.tsx** (350 lines)
   - CRUD UI for managing project sources
   - Form to add: title, authors, year, URL, source type, notes
   - Source list with clickable citations
   - Citation format display (expandable)
   - APA/MLA/Chicago format reference guide
   - Delete with confirmation

2. **CitationTool.tsx** (280 lines)
   - TipTap extension for inline citations
   - Citation toolbar button (search sources by title/author)
   - Keyboard shortcut: Cmd/Ctrl+Shift+C
   - Renders citations as superscript numbered markers [1]
   - Click handlers for editing/viewing source details
   - Drag-to-move, Backspace-to-delete

3. **index.ts** (Component exports)

## Features Implemented

✅ **Source Management**
- Add sources (book, article, website, journal, other)
- Store authors, year, URL, notes
- Pre-computed citation formats (APA, MLA, Chicago, IEEE)
- Delete sources with confirmation

✅ **Citation Insertion**
- TipTap editor integration
- Search bibliography by title/authors
- Insert citations with keyboard shortcut
- Track citation number, position, context
- Superscript rendering [1], [2], etc.

✅ **Database Schema**
- Bibliography table with citation_formats JSON
- Chapter_citations junction with full context
- Indexes for fast lookups
- Cascading deletes when sources/chapters deleted

✅ **API Endpoints**
- Full CRUD for sources (GET, POST, PATCH, DELETE)
- Full CRUD for citations (GET, POST, DELETE)
- Authorization checks (book owner only)
- Listing with pagination (limit <= 500)
