## Architecture

### Data Flow: Beta Reader Matching
1. Author creates book → queued for matching
2. Backend finds beta readers matching profile (genre, word count, capacity)
3. Show author a list of compatible readers
4. Author sends match request → BetaReaderMatch (status: proposed)
5. Beta reader accepts → status: accepted, notifications sent
6. Beta reader provides feedback
7. Author can rate feedback quality

### Data Flow: Writing Groups
1. Author creates group → WritingGroup with creator role
2. Author becomes first member
3. Other authors search groups by genre/stage
4. Join → WritingGroupMember (status: active)
5. Members post content to discussion
6. Group mods moderate content

### Data Flow: Author Messaging
1. Author A sends message to Author B → AuthorMessage created
2. Author B receives notification
3. Author B reads message → is_read=1, read_at set
4. Conversation thread tracked via reply_to_id

### Data Flow: Collaboration
1. Author A requests collab with Author B (co-author, editor, etc.)
2. AuthorCollaboration created (status: proposed, optional book_id)
3. Author B accepts/rejects
4. If accepted → can be linked to chapter editing flows

## API Examples

### Get Author Profile
```bash
GET /api/v1/community/profile/user-id
Response: {
  "id": "...",
  "user_id": "...",
  "bio": "I write sci-fi and fantasy...",
  "website": "myauthor.com",
  "published_books_count": 3,
  "total_readers": 1500,
  "average_rating": 4.3
}
```

### Create Beta Reader Profile
```bash
POST /api/v1/community/beta-reader/profile
{
  "experience_level": "expert",
  "preferred_genres": ["sci-fi", "fantasy"],
  "max_word_count": 120000,
  "max_concurrent_books": 5
}
```

### Request Beta Reader
```bash
POST /api/v1/community/beta-reader/match
{
  "book_id": "book-id",
  "beta_reader_id": "reader-id"
}
Response: {
  "status": "proposed",
  "compatibility_score": 0.87
}
```

### Create Writing Group
```bash
POST /api/v1/community/groups
{
  "name": "Sci-Fi Romance Writers",
  "description": "For authors writing at the intersection of sci-fi and romance",
  "visibility": "public",
  "focus_genres": ["sci-fi", "romance"],
  "writing_stage": ["drafting", "editing"]
}
Response: {
  "id": "...",
  "slug": "sci-fi-romance-writers",
  "member_count": 1
}
```

### Send Message to Author
```bash
POST /api/v1/community/messages
{
  "recipient_id": "author-id",
  "subject": "Collaboration opportunity",
  "content": "Would you be interested in..."
}
Response: {
  "id": "...",
  "is_read": false,
  "created_at": "2024-04-12T..."
}
```
