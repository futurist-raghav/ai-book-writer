## Architecture Notes

### Canvas Rendering
- Hardware-accelerated for performance
- Optimized for 1000+ events
- DPI-aware drawing for different screens
- Efficient event lookup with spatial indexing

### Cycle Detection Algorithm
- DFS-based O(V+E) complexity
- Topological sort for critical path
- Transitive closure for dependency analysis
- Memo-ized for performance

### Keyboard System
- Global event listener with cleanup
- Platform detection (Mac vs Windows)
- Event delegation pattern
- Prevents default browser behaviors

### Export System
- Client-side only (no server call)
- Automatic file download via blob
- Memory-efficient streaming for large datasets
- HTML template with embedded CSS
