# P2.4 Bibliography & Citations Module - Complete Implementation Guide

## Overview

The Bibliography & Citations module (P2.4) provides comprehensive source management and citation insertion capabilities for the Scribe House. Writers can maintain project-wide bibliographies and insert formatted citations directly into chapter text.

**Status:** ✅ **Complete** - All backend routes, database models, frontend components, and tests delivered

**Timeline:** 930 lines of production code across 3 layers (Backend API + Database + Frontend UI)

## Backend Implementation

### Database Schema (Migration 011)

#### Bibliography Table
```sql
CREATE TABLE bibliography (
  id UUID PRIMARY KEY,
  book_id UUID NOT NULL,
  title VARCHAR(500) NOT NULL,
  authors JSONB,                 -- ["Author 1", "Author 2", ...]
  year INTEGER,
  source_type VARCHAR(50),       -- book, article, website, video, etc.
  source_url VARCHAR(2000),
  citation_formats JSONB,        -- {apa: "...", mla: "...", chicago: "..."}
  notes TEXT,
  is_deleted BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP,
  updated_at TIMESTAMP,
  FOREIGN KEY (book_id) REFERENCES books(id) ON DELETE CASCADE,
  INDEX (book_id, title)
);
```

#### ChapterCitations Table
```sql
CREATE TABLE chapter_citations (
  id UUID PRIMARY KEY,
  chapter_id UUID NOT NULL,
  bibliography_id UUID NOT NULL,
  citation_number INTEGER,      -- [1], [2], [3] for inline markers
  page_number VARCHAR(50),      -- Optional: "42" or "42-44"
  context_offset INTEGER,       -- Character position in chapter text
  context_snippet TEXT,         -- 100 chars before/after citation
  citation_format VARCHAR(20),  -- apa, mla, chicago, ieee (default: apa)
  created_at TIMESTAMP,
  updated_at TIMESTAMP,
  FOREIGN KEY (chapter_id) REFERENCES chapters(id) ON DELETE CASCADE,
  FOREIGN KEY (bibliography_id) REFERENCES bibliography(id) ON DELETE CASCADE,
  UNIQUE (chapter_id, bibliography_id, context_offset),
  INDEX (chapter_id, citation_number)
);
```

### Authentication & Authorization

All endpoints require:
- **Bearer Token** in `Authorization` header
- **Book Ownership** - user must be the book author
- **Chapter Membership** - chapter must belong to book

Errors:
- `401 Unauthorized` - Missing or invalid token
- `403 Forbidden` - User is not book author
- `404 Not Found` - Book, chapter, or source doesn't exist

### Citation Extension (TipTap)

**Purpose:** TipTap editor mark for rendering citations

**File:** `/frontend/src/components/bibliography/CitationExtension.ts` (Part of CitationTool)

**Mark Definition:**
```typescript
const CitationMark = Mark.create({
  name: 'citation',
  
  addAttributes() {
    return {
      citationId: { default: null },
      citationNumber: { default: 1 },
      sourceTitle: { default: '' }
    };
  },

  parseHTML() {
    return [{ tag: 'sup[class=citation]' }];
  },

  renderHTML({ node }) {
    return ['sup', { class: 'citation' }, `[${node.attrs.citationNumber}]`];
  },

  addCommands() {
    return {
      insertCitation: (attrs) => ({ commands }) => {
        return commands.setMark(this.name, attrs);
      },
      toggleCitation: (attrs) => ({ commands }) => {
        return commands.toggleMark(this.name, attrs);
      },
      removeCitation: () => ({ commands }) => {
        return commands.unsetMark(this.name);
      }
    };
  },

  addKeyboardShortcuts() {
    return {
      'Mod-Shift-c': () => this.editor.commands.insertCitation(),
    };
  }
});
```

## Citation Formatting

### Pre-Generated Formats

All bibliography sources automatically generate citations in 4 major formats:

**APA (American Psychological Association)**
```
Author(s). (Year). Title. Publisher.
Smith, J., & Doe, J. (2020). Test Book.
```

**MLA (Modern Language Association)**
```
Author(s). "Title." Publisher, Year.
Smith, John, and Jane Doe. "Test Book." 2020.
```

**Chicago Manual of Style**
```
Author(s). "Title." Publisher, Year.
Smith, J., and J. Doe. "Test Book." 2020.
```

**IEEE (Institute of Electrical and Electronics Engineers)**
```
Initials. Surname, "Title," Publisher, Year.
J. Smith and J. Doe, "Test Book," 2020.
```

### Format Generation Algorithm

```typescript
function generateCitationFormats(source: Bibliography): CitationFormats {
  const authors = formatAuthors(source.authors);
  const year = source.year || 'n.d.';
  const title = source.title;
  
  return {
    apa: `${authors} (${year}). ${title}.`,
    mla: `${formatMLAAuthors(source.authors)} "${title}." ${year}.`,
    chicago: `${authors}. "${title}." ${year}.`,
    ieee: `${formatIEEEAuthors(source.authors)}, "${title}," ${year}.`
  };
}
```

### Database Indexes
- `bibliography.book_id` - Fast filtering by project

- ✅ Limit bibliography results to 500 per page
- ✅ Eager load bibliography in citation responses
- ✅ Cache formatted citations in JSON field
- ✅ Use UUID for primary keys (B-tree friendly)

### Response Caching
```typescript
// Cache bibliography per book for 5 minutes
const [cache, setCache] = useState(new Map());

const getCachedBibliography = async (bookId: string) => {
  const cached = cache.get(bookId);
  if (cached && DateDifference(cached.time, now) < 5 * 60000) {
    return cached.data;
  }
  
  const data = await apiClient.bibliography.list(bookId);
  setCache(new Map([...cache, [bookId, { data, time: now }]]));
```

### Authorization Errors on Create

**Problem:** 403 Forbidden when adding source
```
✗ Access denied creating bibliography for book
```

**Solution:**
```tsx
// Verify: Book owner = Current user
if (book.author_id !== currentUser.id) {
  // User doesn't own this book
  throw new Error('Authorization denied');
}
```

### Citation Numbers Out of Sequence

**Problem:** Citations numbered [1], [3], [5] after deletions
```
✗ Citation numbers not renumbered after deletion
```

**Solution:**
```typescript
// After deletion, renumber remaining citations
const citations = await apiClient.bibliography.listChapterCitations(chapterId);
for (let i = 0; i < citations.length; i++) {
  citations[i].citation_number = i + 1;
}
```
